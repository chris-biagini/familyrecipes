# Ingredient Catalog Improvements — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers-extended-cc:executing-plans to implement this plan task-by-task.

**Goal:** Add ingredient aliasing, overhaul bin/nutrition as a USDA-only curation tool, populate ~25-30 diverse catalog entries via trial runs, and validate the schema against real-world recipes.

**Architecture:** Aliases are a JSON array on the existing `ingredient_catalog` table. `lookup_for` builds a reverse map from aliases → canonical entries during the hash merge. `bin/nutrition` drops manual-entry code and adds richer USDA search previews + alias prompting. Trial runs iterate on both the tool and the data.

**Tech Stack:** Rails 8, SQLite, Minitest, USDA FoodData Central API

**Step 3: Implement alias lookup in the model**

In `app/models/ingredient_catalog.rb`, modify `add_ingredient_variants` to also add alias keys:

```ruby
def self.add_ingredient_variants(lookup)
  extras = {}
  lookup.each_value do |entry|
    FamilyRecipes::Inflector.ingredient_variants(entry.ingredient_name).each do |variant|
      extras[variant] = entry unless lookup.key?(variant)
    end
    add_alias_keys(extras, entry, lookup)
  end
  lookup.merge(extras)
end

def self.add_alias_keys(extras, entry, lookup)
  return if entry.aliases.blank?

  entry.aliases.each do |alias_name|
    lowered = alias_name.downcase
    # Canonical names always win over aliases
    next if lookup.key?(alias_name) || lookup.key?(lowered)

    extras[alias_name] = entry unless extras.key?(alias_name)
    extras[lowered] = entry unless extras.key?(lowered)
  end
end
private_class_method :add_alias_keys
```

### Task 8: Add alias prompting after save

**Files:**
- Modify: `bin/nutrition`

**Step 1: Add alias prompt to review_and_save**

After the user saves an entry, prompt for aliases. Suggest candidates based on the USDA description and the ingredient name:

```ruby
def suggest_aliases(name, entry)
  suggestions = []
  usda_desc = entry.dig('sources', 0, 'description') || ''

  # Extract meaningful words from USDA description
  # e.g., "Milk, whole, 3.25% milkfat" → suggest "Whole milk"
  # e.g., "Wheat flour, white, all-purpose" → suggest "All-purpose flour", "White flour"

  # Also suggest without qualifier: "Flour (all-purpose)" → "Flour"
  base, qualifier = name.match(/\A(.+?)\s*\(([^)]+)\)\z/)&.captures || [name, nil]
  suggestions << base if qualifier

  suggestions.uniq - [name]
end

def prompt_aliases(name, entry)
  existing = entry['aliases'] || []
  suggestions = suggest_aliases(name, entry)

  puts "\nAliases (current: #{existing.any? ? existing.join(', ') : 'none'}):"
  suggestions.each { |s| puts "  Suggested: #{s}" }
  puts "  Enter aliases (comma-separated), or Enter to skip:"

  input = $stdin.gets&.strip
  return existing if input.nil? || input.empty?

  new_aliases = input.split(',').map(&:strip).reject(&:empty?)
  (existing + new_aliases).uniq
end
```
