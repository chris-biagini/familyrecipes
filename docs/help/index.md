# familyrecipes

`familyrecipes` is a self-hosted home recipe organizer. 

## Features

- Markdown-based recipe format that is easy to write, read, import, and export
- Graphical recipe editor for people unmoved by the preceding line
- Easy scaling: halve, double, or triple recipes with a click, or type in an
  arbitrary scaling factor
- Per-serving nutrition facts (based on nutrition data you enter)
- Support for crossing off steps and ingredients as you cook
- USDA API integration to make entering nutrition data easy
- Anthropic API integration for AI-assisted recipe import
- Automatic grocery list generation from selected recipes
- Grocery list sharing with live updates
- Recipe cross-linking (e.g., to share one Dough recipe across multiple Pizza
  recipes)
- Authentication via Authelia
- Installation via Docker
- "Anki for groceries" exponential backoff for ingredients (see groceries.md)
