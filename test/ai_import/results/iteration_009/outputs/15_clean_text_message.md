# Guacamole

- Avocados, 2
- Lime, 1/2
- Salt, little bit
- Cilantro, some: Chopped small. Optional.
- Jalapeño, 1/2 or less: Chopped small. Optional.

Mash the avocados with a fork, squeeze the lime in, chop the cilantro and jalapeño real small and mix it all together. Taste it and add more salt/lime if u need to. Don't overmix it should be chunky.
